# CashTrackr

CashTrackr es una aplicación web desarrollada con Laravel 13 para el seguimiento de gastos, ingresos y presupuestos personales.
El objetivo del proyecto es ayudar a los usuarios a organizar sus finanzas de forma simple, rápida y eficiente.

## Características

* Registro de gastos e ingresos
* Gestión de presupuestos
* Categorías personalizadas
* Estadísticas financieras
* Historial de movimientos
* Panel administrativo intuitivo
* Diseño responsive

## Tecnologías utilizadas

* PHP
* Laravel 13
* pgsql
* Blade / Tailwind CSS
* JavaScript

## Instalación

Clonar el repositorio:

```bash
git clone https://github.com/tu-usuario/CashTrackr---Pedev.git
```

Entrar al proyecto:

```bash
cd cashtrackr
```

Instalar dependencias:

```bash
composer install
npm install
```

Configurar variables de entorno:

```bash
cp .env.example .env
```

Generar la clave de la aplicación:

```bash
php artisan key:generate
```

Ejecutar migraciones:

```bash
php artisan migrate
```

Iniciar el servidor:

```bash
php artisan serve
```

## Objetivo del proyecto

CashTrackr nace como un proyecto de aprendizaje y práctica enfocado en el desarrollo backend y frontend con Laravel, aplicando buenas prácticas y funcionalidades reales de administración financiera.
